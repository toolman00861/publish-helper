功能需求清单
输入模块

多行文本框，支持粘贴长文本（微信/小红书/抖音内容）
可选填事件标题（留空则 AI 生成）
一键清空

AI 分析模块（一次 API 调用）
返回结构化 JSON，包含以下字段：
title        // 事件标题
type         // 分类：教育类 / 劳动就业类 / 校园生活类 / 学生权益类 / 校园安全类 / 其他
summary      // 3-5 句核心摘要
score        // 热度分 1-10
reason       // 评分理由（影响人数、传播性、历史参考）
输出模块

页面内渲染 Markdown 格式报告
导出 TXT（浏览器直接下载）
导出 Word（.docx，用 docx 库生成）

体验细节

分析中显示 loading 状态
错误提示（输入为空 / API 报错）
报告生成后支持重新分析
